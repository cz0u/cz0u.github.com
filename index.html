Install GRUB2 onto a Disk Image

  1. loop mount the partitioned disk image:

     # losetup --show --find disk.img
     /dev/loop0

  2. get major and minor numbers of the loop device.

     # ls -l /dev/loop0

  3. create a device-mapper node.

     # echo '0 SIZE linear 7:0 0' |dmsetup
  4.  # kpartx -a /dev/mapper/hda
  5.  # mount /dev/mapper/hda1 /tmp/disk
  6.  # grub-install --boot-directory=/tmp/disk/ /dev/mapper/hda

  7. Cleanup

     # umount /tmp/disk
     # rm -rf /tmp/disk
     # kpartx -d /dev/mapper/hda
     # dmsetup remove hda
     # losetup -d /dev/loop0
