Install GRUB2 onto a Disk Image

来源: http://tomermargalit.wordpress.com/tag/create-grub2-bootable-image-over-loopback-device/

system: fedora 18 grub: GRUB 2.0 0. prepare disk image:

    # touch disk.img && truncate --size 512M disk.img

  1. loop mount the partitioned disk image:

     # losetup --show --find disk.img
     /dev/loop0
  2.  # partx /dev/loop0
     # ls /dev/loop0*
     /dev/loop0 /dev/loop0p1

  3. mount the loop device 

     # mount /dev/loop0p1 /tmp/disk

  4. install grub2

     # grub2-install --boot-directory=/tmp/disk/boot/ /dev/mapper/hda

  5. Cleanup

     # umount /tmp/disk
     # rm -rf /tmp/disk
     # partx -d /dev/loop0
     # losetup -d /dev/loop0
